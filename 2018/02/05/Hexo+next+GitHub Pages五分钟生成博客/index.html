Hexo+next+GitHub Pages五分钟生成博客

Posted on 2018-02-05

需要安装的软件：

  • node.js
  • git

node安装好之后我们使用hexo

此时基本内容都已经安装完毕。

  • 新建文件夹放置博客内容 mkdir blog
  • 进入文件夹内执行初始化 hexo init
  • 安装依赖包 npm install
  • 修改此文件夹下根目录配置文件_config.yml启用主题theme: next

安装next主题

  • 下载主题 git clone https://github.com/iissnan/hexo-theme-next themes/next

开启github page配置

  • 在github建立一个公开放库，用户名.github.io注意一定要保证用户名就是自己github的用户名
  • 在settings里面找到GitHub Pages开启

将博客内容推送至github

  • 修改根目录配置文件_config.yml，查找到deploy如下修改，注意修改自己的地址。

  • 执行命令推送

  • 推送成功

查看网页

  • 打开 用户名.github.io 成功打开博客页面。我自己的如：fiany.github.io。

总结

是不是很简单就可以搭建出来一个博客网站。
